# AI活用型Web制作・提案ポートフォリオ

このポートフォリオは、**AIを活用したWeb制作と提案支援**を中心に、
**情報整理・構成設計・伝え方の支援**まで含めたサービスの紹介を目的としています。

企業のご要望を整理し、読みやすく、納得感のある形で伝えることを大切にしています。

## 概要

このサイトは、次のような価値を伝えるために作成しています。

- AIを使ってWeb制作・LP制作・提案ページを効率よく作ること
- 情報を整理し、ユーザーにとって分かりやすい構成にすること
- 単なる「作る」だけでなく、「提案できる」ことを重視すること

「見た目」だけでなく、**誰に何を伝えるべきか**を整理する力も、
このポートフォリオの核になっています。

## 特徴

- **落ち着いた上品なデザイン**で、クライアントに安心感を与える構成
- **AI活用型の制作フロー**を可視化し、仕組みが伝わるように設計
- **情報整理・コンテンツ設計**を重視し、読みやすさを意識
- **提案力**を示す制作実績を整理し、実務の背景をわかりやすく提示

## 使用技術

- HTML
- CSS
- JavaScript

外部フォントや軽量なデザイン要素を活用し、
高品質な印象を保ちながら、シンプルに構成されています。

## 制作意図

このポートフォリオを作った目的は、次の2点です。

1. **AI活用型Web制作・提案支援の価値を伝えること**
2. **技術だけでなく、情報整理能力と提案力を持つ人材であることを示すこと**

実務経験を通じて培ってきた「整理する力」「伝える力」を、
見える形でまとめることを意識しています。
